Сервис авторизации на Go с использованиеим Postgres. 

Шаги запуска приложения: 
1) склонируйте проект в удобную для вас директорию с помощью команды *git clone https://github.com/muzhevsky/golang_auth_service.git*.
2) выполните команду *bash run.sh* и введите пароль по умолчанию для postgres *passssap* (можно настроить в файле docker-compose.yaml).
3) для проверки успешного запуска контенеров можно выполнить команду *docker ps* (опционально).
4) отправьте тестовый запрос по протоколу http:
   1) Полный адрес запроса: http://\*Хост\*:\*Порт\*/user 
   2) Метод: POST
     Пример тела запроса:
       {
         "login":"testLogin",
         "password":"testPassword123",
         "e-mail":"andreimuzhevsky@yandex.ru",
         "nickname":"testNickname"
       }
   3) В результате должен вернуться ответ в формате { "id" = 1; }, содержит id пользователя в системе.
   
   P.S. Хост - ip-адрес вашей машины, где развернут проект.

    Порт: По умолчанию 8080, настраивается в ./authorization/config/config.toml
